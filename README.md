# close_caption_converter

A Flutter project.

Download the close caption from Bilibili and then convert it to .lrc or .srt. 

A BVid is needed. (Get the BVid from the video link. )

<img width="498" alt="image" src="https://github.com/Biyanci/close_caption_converter/assets/98510207/60e6aaf5-dbbf-4d7b-9e1b-7cdbdc15d927">

Choose a listed caption to convert. (Or tap the right button to read in detail. )

<img width="448" alt="image" src="https://github.com/Biyanci/close_caption_converter/assets/98510207/fd95354f-36c3-4c8b-8ed1-427d9fbeb5a8">

Set the delay to make the caption show later or earlier. The delay must be a number.

After converted, you can edit the file in the Edit view. 

<img width="421" alt="image" src="https://github.com/Biyanci/close_caption_converter/assets/98510207/a0a70b98-e4bb-49f7-85f7-b2eb0b712ecd">

Last step, push the Save button top-right to save the file. At the same time, the directory will be show at the bottom snackbar. 

For Windows, the directory is C:\Users\{Your user name}\Documents.

For Android, the directory is /Android/data/com.example.close_caption_converter/files.
